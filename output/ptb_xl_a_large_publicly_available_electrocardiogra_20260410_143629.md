## Summary

The PTB-XL is a comprehensive, large-scale publicly available electrocardiography (ECG) dataset containing 21,799 clinical 12-lead ECGs from nearly 19,000 patients. This dataset is extremely rich, as recordings were annotated by multiple cardiologists using 71 standardized ECG statements covering diagnostic, form, and rhythm assessments. Beyond the waveforms, it provides extensive metadata on demographics and infarction characteristics, enhancing its value for complex research. Overall, PTB-XL serves as a robust resource designed for training and evaluating advanced automatic ECG interpretation algorithms.

## Key Facts

*   The dataset is called PTB-XL and contains electrocardiography (ECG) data.
*   It consists of 21,799 clinical 12-lead ECGs, each of 10-second length, derived from 18,869 patients.
*   The ECG records were annotated by up to two cardiologists, assigning potentially multiple ECG statements to each.
*   The annotations cover 71 different ECG statements conforming to the SCP-ECG standard, including diagnostic, form, and rhythm statements.
*   The raw waveform data was initially collected with Schiller AG devices between October 1989 and June 1996.
*   The dataset includes extensive metadata on demographics (age, sex, weight, height) and infarction characteristics.
*   The patient demographic distribution is 52% male and 48% female, with ages ranging from 0 to 95 years.
*   The dataset ensures data privacy by pseudonymizing personal information and shifting ECG recording dates by a random offset per patient.
*   The structure includes separate directories for records with 100 and 500 measurements, containing `.dat` and `.hea` files.
*   The underlying waveform files are stored in the WaveForm DataBase (WFDB) format with 16 bits.
